### Usage
At first you need to install dependencies with npm.

`npm i`

To start app:

`npm run build`

and then

`npm run start`

Local app will be listening on `http://127.0.0.1:3060`

For enabling GUI

`npm run start-with-gui`

GUI is served at `http://127.0.0.1:3070`

### Development
To start app development:

`npm run dev`

### Test
To run tests:

`npm run test`
